### アプリのデモ・実行はこちら

**[統計検定2級対策アプリを開く](https://yusuke223119.github.io/Quiz/)**

## 📂 プロジェクト構造

```text
/ (Root)
├── index.html          # TOP画面（ジャンル選択）
├── quiz.html           # 問題回答画面
├── explanation.html    # 解説一覧画面
├── main.js             # メニュー制御ロジック
├── menu_data.js        # ジャンル・メニュー定義データ
├── pictures/           # 画像リソース
└── questions/          # 問題データ
```
